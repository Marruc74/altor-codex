# Reference — source material

Working notes, summaries and OCR transcriptions of the *Drakar och Demoner* /
Ereb Altor sourcebooks and adventures that the codex content is built from.
These are not part of the app build (nothing here is imported by Vite); they
live in the repo so the source material is kept in one place for further use.

Filenames use the canonical Swedish book title, folded to plain ASCII
kebab-case. English glosses are below. Where a book has more than one file, a
suffix marks the variant (`-characters`, `-detailed`, `-entities`, `-summary`).
Two adventures were captured twice; the larger file is primary and the other is
suffixed `-alt`.
